### Overview

이 프로젝트는 BIM 데이터를 뼈대로 삼고, DataFaker로 살을 붙인 뒤,
LLM(대규모 언어 모델)을 인터페이스로 활용하여 각 공정 단계의 의사결정을 돕는
지능형 건설 시뮬레이션 샌드박스입니다.
복잡한 수치 대신 "질의응답"이라는 직관적인 방식으로 도메인 지식을 추출하는 것이 핵심입니다.

### Requirements

BIM-Centric Domain Sandbox
핵심 메커니즘: DataFaker를 통한 가상 데이터 생성 + LLM 기반의 단계별 질의응답(Q&A) 시스템.
최종 목적: 데이터에 기반한 직관적인 의사결정 보조 도구(Decision Support Tool) 구현.
